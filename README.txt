This repository contains the code for a computerized version of the Wisconsin Card Sorting Task (WCST). The INDEX.HTML file contains code to generate a homepage where people can choose either a classic game type or an arcade game type. The CLASSIC.HTML file contains code to generate a classic WCST. The ARCADE.HTML file contains code to generate an Arcade WCST type with different levels. 

All pages rely on the styles.css document for the lay-out of the pages

For debugging purposes, the previous rule and current rule can be logged in the console. To do so, decomment the debugInConsole() function on rule 116 and 131 of the game.js script. 

Not all game animations are fully supported by Internet Explorer and Microsoft Edge. Please use another browser for the full experience. 
In case of errors, please report the issues at the issue page of the project.
